# 예제 지도 {#example-map}





예제들을 큰 범주 → 작은 범주로 묶어두었습니다. 필요한 부분만 골라 보세요.




- HUD (화면 위 표시)                            좌표, 경험치 바, 아이콘 알림
- 핫바, 아이템 내구도, 상태 표시



서버 폼/대화창

- 버튼 목록(form_buttons) 기반 UI
- 제목(title_text)로 화면 분기



NPC 상호작용

- live_player_renderer로 NPC 표시
- 대화 + 선택지 버튼 구성



인벤토리/컨테이너

- 상자, 제작대, 화로 같은 슬롯 화면
- 아이템 아이콘, 스택 수, 선택 강조



채팅/알림

- 채팅 필터, 자동완성 버튼
- 액션바/알림 텍스트 분해



설정/메뉴

- 토글/슬라이더/드롭다운
- 섹션 분리, 탭 이동



월드/시작 화면

- 월드 목록, 서버 목록
- 버튼 그리드, 카드형 레이아웃



상점/마켓

- 상품 카드, 배너, 프로모션
- 스크롤 목록, 필터/정렬



스킨/프로필

- 미리보기 렌더러
- 선택/잠금 상태 표시



로그인/안전/경고

- 확인/취소 버튼, 경고 메시지
- 접근성 안내 및 링크



애니메이션/연출

- 로딩 오버레이, 진행 바
- 버튼 강조, 페이드 효과






처음엔 HUD나 서버 폼 하나만 골라서 따라 하는 것이 가장 쉽습니다.
