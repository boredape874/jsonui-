## 기초 준비 {#setup}





처음 시작할 때 필요한 것들을 간단하게 정리했습니다.







            ### 리소스 팩

JSON UI는 리소스 팩 안에 넣어야 게임에서 읽습니다.





            ### UI 폴더

UI 파일은 보통 `ui` 폴더에 둡니다.





            ### 화면 파일

HUD, 서버 폼, 인벤토리 같은 화면마다 파일이 다릅니다.





            ### 테스트 습관

한 번에 많이 바꾸지 말고, 한 줄씩 바꾸고 바로 테스트합니다.







          처음에는 HUD 화면만 수정해 보는 것이 가장 빠르게 결과를 확인하는 방법입니다.





아주 기본적인 파일 뼈대


```
{
  // 네임스페이스: 이 파일의 이름표
  "namespace": "hello",

  // 화면에 넣을 패널
  "hello_root": {
    "type": "panel",
    "size": ["100%", "100%"]
  }
}
```
