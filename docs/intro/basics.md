## 기초 개념 {#basics}





처음 보는 사람도 이해할 수 있도록 쉬운 말로 정리했습니다.







            ### JSON UI는 “화면 설계도”

텍스트 파일에 화면의 위치, 크기, 버튼을 적어두는 방식입니다.





            ### 컨트롤은 “레고 블록”

panel, label, button 같은 블록을 쌓아서 화면을 만듭니다.





            ### 바인딩은 “연결선”

게임 데이터와 화면을 이어주는 선입니다.








| 아주 쉬운 설명 | 실제 용어 | 예시 |
| --- | --- | --- |
| 투명한 상자 | panel | 다른 컨트롤을 담는 그릇 |
| 스티커 글씨 | label | 텍스트 출력 |
| 그림 판넬 | image | 텍스처 표시 |
| 스위치 | button | 클릭 동작 |








            ### 읽는 순서

- namespace: 큰 폴더 이름
- type: 어떤 블록인지
- size/anchor: 크기와 위치
- controls: 안에 들어갈 것들






            ### 초보자 실수 TOP 3

- 쉼표 하나 누락
- 따옴표를 빼먹기
- controls 배열을 비워둠








          JSON UI는 주석을 사용할 수 있습니다. `//`로 왜 이렇게 작성했는지 설명을 남겨두면 좋아요.
