# 소개 {#intro}

<div class="alert-box alert-info">
이 가이드는 <strong>프로그래밍을 전혀 해보지 않은 완전 초보자</strong>를 위해 작성되었습니다.
모든 용어와 개념을 처음부터 설명하며, 천천히 따라하면 누구나 JSON UI를 만들 수 있습니다!
</div>

<h3>JSON UI가 뭔가요?</h3>
<p>
마인크래프트 베드락 에디션(모바일, 윈도우10, 콘솔 등)은 게임 화면을 <strong>JSON UI</strong>라는 시스템으로 만듭니다.
이것을 수정하면 게임 화면의 모양을 바꿀 수 있습니다!
</p>

<div class="term-box">
<div class="term-title">JSON UI로 할 수 있는 것들</div>
<div class="term-content">
<ul>
<li><strong>HUD 커스터마이징</strong> - 체력바, 허기바, 핫바(단축바)의 모양과 위치 변경</li>
<li><strong>새로운 UI 요소 추가</strong> - 속도계, 경험치 바, 커스텀 텍스트 등 추가</li>
<li><strong>서버 폼 디자인</strong> - 서버에서 보내는 메뉴 화면을 예쁘게 꾸미기</li>
<li><strong>애니메이션 효과</strong> - 움직이는 UI, 페이드 인/아웃 효과 등</li>
<li><strong>데이터 표시</strong> - 명령어로 보낸 데이터를 화면에 표시</li>
</ul>
</div>
</div>

<div class="alert-box alert-warning">
<strong>알아두세요:</strong> 최신 마인크래프트 버전에서는 일부 화면이 <strong>Ore UI</strong>로 바뀌었습니다.
Ore UI는 하드코딩되어 있어서 수정할 수 없지만, HUD와 같은 중요한 화면은 여전히 JSON UI로 수정 가능합니다!
</div>

<h3>이 가이드의 특징</h3>
<div class="lecture-note">
<div class="lecture-note-title">이렇게 배웁니다</div>
<ul>
<li> <strong>모든 용어를 쉬운 말로</strong> 설명합니다</li>
<li> <strong>코드마다 주석(설명)</strong>을 달아서 무엇을 하는지 알려줍니다</li>
<li> <strong>단계별로 천천히</strong> 진행합니다</li>
<li> <strong>실제 예제</strong>를 직접 따라 만들 수 있습니다</li>
<li> <strong>초급부터 고급까지</strong> 모든 내용을 다룹니다</li>
</ul>
</div>
