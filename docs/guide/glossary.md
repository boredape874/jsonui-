# 용어 사전 {#glossary}

> 난이도: 초급

<div class="term-box">
<div class="term-title">A</div>
<div class="term-content">
<p><strong>ActionForm</strong> - 버튼 목록이 있는 서버 폼. NPC 대화 등에 사용</p>
<p><strong>Alpha</strong> - 투명도. 0.0 = 완전 투명, 1.0 = 불투명</p>
<p><strong>Anchor</strong> - 앵커. 요소를 배치할 기준점 (9개 위치)</p>
<p><strong>Animation</strong> - 애니메이션. UI 요소에 움직임을 주는 기능</p>
</div>
</div>

<div class="term-box">
<div class="term-title">B</div>
<div class="term-content">
<p><strong>Binding</strong> - 바인딩. 게임 데이터를 UI에 연결</p>
<p><strong>Button</strong> - 버튼. 클릭 가능한 UI 요소</p>
<p><strong>Button Mapping</strong> - 버튼 매핑. 버튼 클릭 시 실행될 동작 정의</p>
</div>
</div>

<div class="term-box">
<div class="term-title">C</div>
<div class="term-content">
<p><strong>Control</strong> - 컨트롤. UI 요소의 다른 이름</p>
<p><strong>Custom Form</strong> - 커스텀 폼. ModalForm의 다른 이름</p>
</div>
</div>

<div class="term-box">
<div class="term-title">E</div>
<div class="term-content">
<p><strong>Element</strong> - 요소. UI를 구성하는 개별 부품</p>
<p><strong>Easing</strong> - 이징. 애니메이션의 속도 변화 곡선</p>
</div>
</div>

<div class="term-box">
<div class="term-title">F</div>
<div class="term-content">
<p><strong>Factory</strong> - 팩토리. 같은 구조의 UI를 반복 생성하는 패턴</p>
</div>
</div>

<div class="term-box">
<div class="term-title">H</div>
<div class="term-content">
<p><strong>HUD</strong> - Head-Up Display. 게임 중 항상 표시되는 UI</p>
</div>
</div>

<div class="term-box">
<div class="term-title">L</div>
<div class="term-content">
<p><strong>Label</strong> - 라벨. 텍스트를 표시하는 요소</p>
<p><strong>Layer</strong> - 레이어. 요소의 깊이 (높을수록 위에 표시)</p>
</div>
</div>

<div class="term-box">
<div class="term-title">M</div>
<div class="term-content">
<p><strong>ModalForm</strong> - 모달 폼. 입력 필드와 슬라이더가 있는 서버 폼</p>
<p><strong>Modification</strong> - 수정. 기존 UI에 새 요소를 추가하는 방법</p>
</div>
</div>

<div class="term-box">
<div class="term-title">N</div>
<div class="term-content">
<p><strong>Namespace</strong> - 네임스페이스. UI 파일의 식별자</p>
</div>
</div>

<div class="term-box">
<div class="term-title">O</div>
<div class="term-content">
<p><strong>Offset</strong> - 오프셋. 앵커에서 얼마나 떨어질지 [x, y]</p>
<p><strong>Operator</strong> - 연산자. 계산이나 조건 비교에 사용 (+, -, =, not 등)</p>
</div>
</div>

<div class="term-box">
<div class="term-title">P</div>
<div class="term-content">
<p><strong>Panel</strong> - 패널. 다른 요소를 담는 컨테이너</p>
<p><strong>Property</strong> - 프로퍼티. 요소의 속성 (size, color 등)</p>
<p><strong>Property Bag</strong> - 프로퍼티 백. 로컬 변수 저장소</p>
</div>
</div>

<div class="term-box">
<div class="term-title">S</div>
<div class="term-content">
<p><strong>Server Form</strong> - 서버 폼. 스크립트로 열리는 UI 창</p>
<p><strong>Stack Panel</strong> - 스택 패널. 자식 요소를 자동으로 정렬하는 패널</p>
</div>
</div>

<div class="term-box">
<div class="term-title">T</div>
<div class="term-content">
<p><strong>Toggle</strong> - 토글. ON/OFF 스위치</p>
<p><strong>Type</strong> - 타입. 요소의 종류 (panel, label, image, button 등)</p>
</div>
</div>

<div class="term-box">
<div class="term-title">V</div>
<div class="term-content">
<p><strong>Variable</strong> - 변수. $ 기호로 시작하는 재사용 가능한 값</p>
<p><strong>Visible</strong> - 보임/안보임. true = 보임, false = 안 보임</p>
</div>
</div>
