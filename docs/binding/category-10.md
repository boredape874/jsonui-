<summary><strong>#author_text_box_item_name</strong> - string (문자열)</summary>

**데이터 타입:** string (문자열)

**간단 설명**
아이템의 이름

**자세한 설명**
이 바인딩은 인벤토리나 컨테이너에 있는 아이템의 이름을 제공합니다.

마인크래프트의 아이템 이름(예: "다이아몬드 검", "떡갈나무 원목" 등)을
현재 게임 언어로 표시합니다.

대장간 모루나 인벤토리 화면에서 자주 사용됩니다.

**실제 사용 예**
예: 인벤토리 툴팁에 아이템 이름 표시, 제작 재료 이름 표시

**코드 예제**
```jsonc
{
  "item_name_label": {
    "type": "label",
    "text": "#item_name",
    "bindings": [
      {
        "binding_name": "#item_name",
        "binding_name_override": "#text",
        "binding_type": "global"
      }
    ]
  }
}
```

**사용 화면**
book screen (총 1개 파일)
